# 🕵️ CyberDetectiveX

CyberDetectiveX is a Python-based CLI cybersecurity investigation simulator that replicates real-world digital forensics and SOC analysis scenarios.

It is designed as an interactive learning project where users act as cyber investigators to solve digital crime cases.

---

## 🎯 Features

- CLI-based interactive gameplay
- Cybercrime case investigation system
- Log analysis and clue detection
- Decision-based story progression
- Beginner-friendly cybersecurity simulation

---

## 🛠️ Tech Stack

- Python 3
- Command Line Interface (CLI)

---

## 📌 Skills Learned

- Ethical Hacking fundamentals
- SOC (Security Operations Center) workflow
- Digital Forensics (DFIR basics)
- Problem-solving & logical reasoning
- Cyber investigation techniques

---
